# Ledger Wallet Desktop

## Requirements

* nodejs v8.x (https://nodejs.org/en/)
* yarn latest (https://yarnpkg.com/fr/docs/install)

## Installation

```
yarn
```

## Development

```
yarn start
```

## Build

> Not package for distribution

```
yarn dist:dir
```

> Package everything

```
yarn dist
```

## Release

```
yarn release
```
